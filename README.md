# AriaDocs Laravel - Documentation Template

This project is an extension of the [AriaDocs](https://github.com/nisabmohd/Aria-Docs) template by [nisabmohd](https://github.com/nisabmohd). Use it if you want a simple PHP-powered documentation and/or blog that can be hosted on any v-server or even shared hosting.

## Features
- Admin Dashboard
- Markdown editor with live preview
- drag-and-drop interface to order and categorise the documentation
